#装饰者模式
##定义与类型
- 定义: 在不改变原有对象的基础上，将功能附加到对象上。提供了比继承更有弹性的替代方案（扩展原有对象功能）
- 类型: 结构型

##适用场景
- 扩展一个类的功能或者添加附加职责
- 动态的给一个对象添加功能，也可以动态的撤销
- 

##优点
- 继承的有力补充，比继承灵活，不改变原有对象的情况下给一个对象扩展功能
- 通过使用不同装饰类或者这些装饰类的排列组合，可以实现不同的效果
- 符合开闭原则

##缺点
- 增加了类和程序的复杂性
- 动态装饰时、多层装饰时会更复杂

- 和适配器模式
